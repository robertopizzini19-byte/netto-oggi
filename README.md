# NettoOggi — Calcolatore Stipendio Netto Italiano

Tool SEO multi-categoria per calcolare lo stipendio netto in Italia. Copre le principali categorie professionali con calcoli specifici per contratto dipendente e regime forfettario.

## Categorie coperte
- Dipendente privato (RAL → netto)
- Partita IVA forfettaria
- Infermiere, Ingegnere, Commesso, Rider
- Cameriere, Operaio, Insegnante, Commercialista

## Tech
- HTML/CSS/JS puro — zero dipendenze
- SEO ottimizzato per ogni categoria
- Deploy su Netlify (drag & drop)

## Deploy
Trascina la cartella su [netlify.com/drop](https://netlify.com/drop).
